# 한반도 해조류 포럼 홈페이지

제공 자료를 바탕으로 제작한 정적 홈페이지입니다.

## 페이지 구성

- `index.html`: 홈
- `about.html`: 소개
- `purpose.html`: 창립 취지
- `forum.html`: 포럼 활동
- `ebook.html`: 학술자료 및 E-book
- `bylaws.html`: 정관
- `research.html`: 해조류와 한반도
- `cooperation.html`: 남북협력
- `news.html`: 뉴스·공지
- `contributions.html`: 기고문
- `join.html`: 회원가입 및 후원
- `contact.html`: 문의

## 폴더 구성

- `css/style.css`: 전체 스타일
- `css/bylaws.css`: 정관 페이지 스타일
- `js/main.js`: 모바일 메뉴 및 스크롤 애니메이션
- `images/1.jpg`: 메인 배경 이미지
- `assets/ebook.pdf`: 한반도해조류포럼 e-book PDF

## 확인 방법

`index.html`을 브라우저로 열면 바로 확인할 수 있습니다.
